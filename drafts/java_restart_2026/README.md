# java_restart_2026

Учебный проект для повторения Java. В репозитории лежат небольшие примеры классов, наследования данных, enum, массивов, списков и вывода информации в консоль.

## Состав проекта

- `Day.java` - основной запускаемый пример. Содержит enum `Tag`, классы записей дневника, работу с массивами тегов, `ArrayList` и фильтрацию записей по тегу.
- `Automobile.java` - базовый класс автомобиля с полями скорости, максимальной скорости, ускорения и торможения.
- `1.ipynb` - notebook с черновыми или учебными материалами.

## Требования

- JDK 17 или новее.

Проверить установленную Java:

```bash
java -version
javac -version
```

## Запуск

Скомпилировать и запустить основной пример:

```bash
javac Day.java
java Day
```

Если нужно скомпилировать все Java-файлы в проекте:

```bash
javac *.java
```

## Что делает `Day`

При запуске программа:

- печатает приветственное сообщение;
- считает примерный процент прогресса года;
- выводит теги с названием длиннее пяти символов;
- создает список видео-записей;
- печатает информацию о записи;
- фильтрует записи по тегу `FOOTBALL`.

## Заметки

Файлы `*.class` создаются компилятором Java и не являются исходным кодом. Их можно удалить и пересоздать командой `javac`.
